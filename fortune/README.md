# 🔮 오라클 v2.0 — 한국식 무당점 + 과학결합 통합 시스템

전통 동양 역술(사주·주역·토정·궁합) + **한국 무당점**(신살·당사주·삼재·납음·60일주·부적) + 서양 점술(타로) + **과학적 결합**(MBTI·바이오리듬·수비학·베이지안 확률) 통합 운세 시스템.

브라우저 단독 동작(서버 불필요), 모바일 친화 다크 UI, 한국인 특화.

## 빠른 시작

```bash
cd fortune
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080
```

## 핵심 알고리즘

### 1. 사주팔자

- **진태양시 보정**: 한국 표준시(135°E)와 출생지 경도 차이를 4분/° 보정. 서울 127°E → -32분.
- **년주**: 입춘(立春, 황경 315°) 기준 — 입춘 이전 출생은 전년 사주.
- **월주**: 12절기(節氣) 기반 월지 결정 + 오호둔(五虎遁)으로 월간 산출.
- **일주**: 1900-01-01 = 庚子日(36) 기준 60갑자 순환.
- **시주**: 2시간 단위 12지지 + 오자둔(五子遁) 시간 산출 (子시는 23:00~01:00).
- **십신**: 일간 기준 오행+음양 관계로 비견/겁재/식신/상관/편재/정재/편관/정관/편인/정인.
- **대운**: 양남음녀 순행 / 음남양녀 역행. 출생일과 다음/이전 절기까지의 일수 ÷ 3 = 대운수.

### 2. 24절기 (천문 계산)

Jean Meeus, *Astronomical Algorithms* 2nd ed. Ch.25(태양 위치) + Ch.27(춘분/하지) 알고리즘 단순화 버전. 태양 황경이 15°의 배수에 도달하는 순간을 이분법으로 탐색. 정확도: 분 단위.

### 3. 주역 점법

- **시초점**(蓍草占, Yarrow): 50개 시초 → 49개 분할(3회) → 1효. 6회 반복. 확률: 노음 1/16, 소양 5/16, 소음 7/16, 노양 3/16.
- **동전점**(三錢): 3개 동전 합으로 1효. 확률 균등.
- 동효(6 또는 9)가 있으면 변괘(變卦) 산출 → 두 괘로 흐름 해석.

### 4. 궁합

다차원 가중 평균:
- 일간 오행 관계 (35%) - 상생/상극/비화
- 일지(부부궁) 합·충 (35%)
- 띠(년지) 합·충·삼합·육합 (15%)
- 오행 보완도 (15%)

## 🪔 한국 무당점 모듈 (v2.0 신규)

한국인 특화 운세 시스템 — 일반 사주에 머무르지 않고 한국 무속 전통의 핵심을 자동화:

### 신살 자동 판별 (12종)
- **길신**: 천을귀인(天乙貴人), 문창귀인(文昌貴人), 천덕귀인, 월덕귀인
- **중성살**: 도화살(桃花殺), 역마살(驛馬殺), 화개살(華蓋殺)
- **흉살**: 양인살(羊刃殺), 백호대살(白虎大殺), 괴강살(魁罡殺)
- **관계살**: 원진살(怨嗔殺), 귀문관살(鬼門關殺)
- **허무살**: 공망(空亡) - 60갑자 공망표 자동 산출

### 60갑자 일주 정밀 해석 (60개 전부)
각 일주마다 본성·연애·직업·재물·건강 + 무당식 풀이. 예시:
- **갑자**: "하늘이 갑목을 자수에 심었으니 머리는 뛰어나도 마음이 외롭다. 늦장가·늦시집 운이로다."
- **병오**: "태양이 정오에 떠 천지를 태우니 권세가 빛난다. 다만 칼이 보이는 직업이 길하다. 화를 다스리라."

### 삼재(三災) 자동 판별
띠 기준 9년 주기. 들/중/날 삼재 + 각 단계별 부적·액땜·주의사항 처방.

### 당사주 12천성
초년·청년·중년·말년 4단계 운세를 12천성(천귀·천액·천권·천파·천간·천문·천복·천역·천고·천인·천예·천수)으로 해석.

### 한국 부적 10종
입춘대길·삼재부·칠성부·호신부·재수부·부부화합부·학업부·퇴마부 등 — 사용 시기·붙이는 위치 안내.

### 띠궁합 144 매트릭스
한국 전통 12지 동물 궁합 빅데이터. 삼합(천생연분 90+점) ↔ 원진(20~35점) ↔ 충(20~30점) 자동 산출.

### 무당식 말투 사전
"신령님이 보여주시는 바는 이러하다", "도화가 피니 인기가 따르나 이성 문제를 조심해야 한다" 식의 한국 무당 어법 자동 생성.

### 납음오행(納音五行) 60종
일주 60갑자별 납음 매핑 (해중금·노중화·대림목·노방토·검봉금·산두화·…) + 물상 해석.

## 🔬 과학 결합 모듈 (v2.0 신규)

전통 해석에 **검증 가능한 정량 지표**를 결합. 모두 결정론적(같은 입력→같은 출력).

### 1. MBTI 16유형 매핑
사주의 오행 분포·십신 구조를 휴리스틱으로 MBTI 4축에 매핑:
- **E/I** ← 화목(양에너지) vs 수금(음에너지)
- **N/S** ← 목화(직관) vs 토금(현실)
- **T/F** ← 일간 양음 + 정관/정인 강도
- **J/P** ← 관성·인성(체계) vs 식상·재성(자유)

신뢰도(0~100%)도 같이 표시. ⚠ 학술적 검증 안된 휴리스틱(재미·참고용).

### 2. 베이지안 운세 확률
"길/흉" 단언 대신 **확률 + 신뢰구간**:
- 형충회합·신살·오행균형 등 시그널 누적
- log-odds → sigmoid → 0~100% 확률
- 시그널 개수 기반 신뢰구간

### 3. 바이오리듬 4사이클
출생일부터의 일수를 sin 파동으로 변환:
- 신체(體) 23일 / 감정(情) 28일 / 지성(知) 33일 / **직관(覺) 38일** (한국식 추가)
- 현재 위상 + 향후 30일 차트 시각화

### 4. 수비학 (Life Path Number)
생년월일 자리수 합산 → 1~9 또는 11/22/33(마스터 넘버). 검증 케이스: 1990-05-15 → 3 ✓

### 5. 종합 점수
사주 60% + 바이오리듬 25% + 수비학 15% 가중 평균. S/A/B/C/D 등급화.
