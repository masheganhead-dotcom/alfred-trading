# Alfred Shader 사용 가이드 (한국어, 비개발자용)

> "내가 해줘"라고 하셨는데, **딱 마우스 클릭 5번**만 직접 해주시면
> 됩니다. 나머지는 다 만들어 놓았어요.

## 0. 이게 뭐예요?

Ableton Live에서 음악 만들 때 **소리에 반응하는 art 비주얼**이 별도
창에 떠 있는 도구입니다. OBS로 그 창 캡처해서 그대로 TikTok/유튜브에
올릴 수 있어요. Phase A에는 3가지 프리셋이 있어요:

- 🪞 **Liquid Chrome** — 음악따라 모양 변하는 무지개색 크롬 덩어리
- 🫧 **Audio Blob** — FFT 따라 표면 일그러지는 유기 blob
- 📺 **VHS Glitch** — 옛날 비디오 테이프 글리치 (RGB split, scan line)

## 1. 준비물 — 본인이 갖고 있어야 하는 것

1. **Mac 또는 Windows 컴퓨터**
2. **Ableton Live 12 Suite** ← *이거 있어야 Max가 따라옴*
   - Standard / Intro 버전은 Max 별도 구매 필요 ($399)
   - Live Suite 안에 Max for Live가 무료로 포함돼 있음
3. (Mac만) **무선 마우스 / 터치패드** (드래그 작업 있음)

> Live 12 Suite가 없으시면? Ableton 공식에서 **30일 무료 평가판**
> 받을 수 있어요 → [ableton.com/en/trial/](https://www.ableton.com/en/trial/)
> Suite로 받아야 Max 포함됩니다.

## 2. 파일 받기 (1분)

이 GitHub 페이지에서 파일들 다운로드:

[m4l/devices/alfred-shader/](https://github.com/masheganhead-dotcom/alfred-trading/tree/claude/max-for-live-plugin-research-bUhx7/m4l/devices/alfred-shader)

GitHub 페이지 오른쪽 위 초록색 **Code** 버튼 → **Download ZIP**.
압축 풀면 `alfred-shader` 폴더가 나와요. 이 안에 필요한 파일이 다 있음.

폴더 안:
```
alfred-shader/
├── AlfredShader.maxpat    ← 디바이스 본체 (이거 열게 됨)
├── alfred_shader.js       ← 보조 코드
├── shaders/               ← 비주얼 셰이더 3개
│   ├── liquid-chrome.jxs
│   ├── audio-blob.jxs
│   └── vhs-glitch.jxs
└── README.md
```

## 3. 폴더를 올바른 위치에 넣기 (30초)

**Mac**:
이 폴더를 통째로 `~/Documents/Max 8/Library/` 안에 넣어주세요.

(찾아가는 법: Finder → 메뉴바의 **이동** → **홈** → **Documents** →
**Max 8** → **Library**. `Max 8` 폴더가 없으면 그냥 `~/Documents/` 안에
넣어도 됩니다.)

**Windows**:
`C:\Users\<본인이름>\Documents\Max 8\Library\` 안에 폴더 통째로 넣기.

## 4. Live 12 안에서 Max 열기 (마우스 클릭 1번)

1. Ableton Live 12 실행.
2. 빈 오디오 트랙 하나 만들기 (없으면 `Cmd/Ctrl + T`).
3. 화면 왼쪽 브라우저에서 **Max for Live → Audio Effect** 폴더 열기.
4. **Max Audio Effect** (빈 디바이스)를 그 오디오 트랙에 드래그.
5. 트랙에 들어간 디바이스의 **연필 모양 ✏️ 아이콘** 클릭 → Max가 열림.

## 5. 패치 파일 열기 (클릭 2번)

Max가 열린 후:

1. Max 메뉴바 → **File** → **Open…**
2. 다운받았던 `alfred-shader/AlfredShader.maxpat` 선택 → 열기.

화면에 박스들이랑 줄들 가득한 패치가 보일 거예요. 이게 디바이스의
내부 구조. **여기서 아무것도 건드릴 필요 없음.** 그대로 다음 단계로.

## 6. 저장하기 (클릭 1번 — 이게 마법의 클릭) 🎩

1. Max 메뉴바 → **File** → **Save As…**
2. 파일 형식 (Format) 드롭다운 → **Max for Live Device (.amxd)** 선택.
3. 디바이스 타입 묻는 창 뜨면 → **Audio Effect** 선택.
4. 저장 위치는 Ableton User Library 안의 Audio Effects 폴더:
   - Mac: `~/Music/Ableton/User Library/Presets/Audio Effects/Max Audio Effect/`
   - Windows: `Documents\Ableton\User Library\Presets\Audio Effects\Max Audio Effect\`
5. 파일명 `Alfred Shader` 같은 거로 저장.

**끝. 이게 다예요.**

## 7. 사용하기

Ableton에서:
1. 브라우저 → **User Library → Audio Effects → Max Audio Effect →
   Alfred Shader** 찾기.
2. 음악이 재생되는 오디오 트랙에 드래그.
3. 디바이스의 **OPEN VISUAL** 큰 파란 버튼 클릭.
4. **별도 창**이 떠요. 음악 재생하면 비주얼이 반응함.
5. **PRESET** 메뉴에서 3가지 중 선택:
   - Liquid Chrome / Audio Blob / VHS Glitch
6. **Intensity / Color / Reaction** 다이얼로 조정.

## 8. TikTok / Reels에 올리려면

별도 창을 OBS Studio로 캡처:
1. OBS Studio 설치 ([obsproject.com](https://obsproject.com/), 무료).
2. OBS에서 **Sources → +  → Window Capture** → Alfred Shader 창 선택.
3. **Start Recording** → Ableton에서 음악 재생.
4. 녹화한 mp4를 TikTok 앱에 올리기 (9:16 비율로 만들고 싶으면 OBS의
   캔버스 해상도를 1080×1920으로).

## ❓ 자주 막히는 부분

### "Max에서 객체가 빨간색 점선으로 보여요"
→ 셰이더 .jxs 파일을 Max가 못 찾는 거예요. 폴더가 Max의 Search Path
에 없는 경우. **Max 메뉴 → Options → File Preferences → Search Path
→ Add Path** 누르고 `alfred-shader` 폴더 추가.

### "OPEN VISUAL 눌렀는데 창이 안 떠요"
→ Max 콘솔(메뉴 → Window → Max Console)에 에러가 떴을 거예요.
빨간 글씨로 뭐가 나오는지 스크린샷 찍어서 알려주세요. 보통은 GPU
드라이버나 Jitter 설정 문제.

### "창은 떴는데 검은 화면이에요"
→ `shaders/` 폴더가 같이 따라가지 않은 경우. `alfred-shader` 폴더
전체를 통째로 Max Library에 넣었는지 확인. 또는 Search Path 추가.

### "그냥 다 모르겠어요"
→ 막힌 화면 스크린샷 찍어서 보내주시면 즉시 어디가 문제인지 보고
설명해드릴게요. 한 단계씩 같이 진행할 수 있어요.

## 정리: 진짜 본인이 해야 하는 건

1. ZIP 다운로드 → 폴더 풀어서 Max Library에 넣기 (1분)
2. Live에서 빈 Max Audio Effect 드래그 → 연필 아이콘 클릭 (10초)
3. **File → Open → AlfredShader.maxpat** (5초)
4. **File → Save As → .amxd → Audio Effect 선택 → 저장** (10초)

**합쳐서 90초**. 진짜 이게 다예요.

막히는 부분만 스크린샷 보내주시면 한 단계씩 같이 갑시다.
